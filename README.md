# 🌍 Global IT Jobs & Salary Analysis Dashboard

## 📌 Project Overview

This Power BI dashboard analyzes the global IT workforce using interactive visualizations to identify hiring trends, salary distribution, workforce composition, and country-wise employee distribution.

---

## 📊 Dashboard Preview

![Dashboard](Images/Dashboard_Preview.png)

---

## 🎯 Key Business Questions Answered

- Which IT category has the highest workforce?
- How has hiring changed over the years?
- How does salary vary across experience levels?
- Which countries have the highest IT workforce?
- How are employees distributed across job categories?

---

## 📈 Dashboard Features

- KPI Cards
- Hiring Trend Analysis
- Job Category Distribution
- Salary Analysis
- Interactive Filtering
- Country-wise Workforce Map

## 🛠 Tools Used

- Microsoft Power BI

- Data Visualization
